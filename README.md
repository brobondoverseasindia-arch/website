# BroBond Overseas

Premium Industrial Safety Gloves Manufacturer & Exporter.

## Project info

This project is the frontend and backend for BroBond Overseas, built with modern web technologies.

## Tech Stack

- **Frontend**: React, Vite, TypeScript
- **UI**: Tailwind CSS, Shadcn UI
- **Backend**: Convex
- **State Management**: React Query, Convex React

## Getting Started

1. **Install dependencies**:
   ```bash
   pnpm install
   ```

2. **Run Development Server**:
   ```bash
   bun run dev:all
   ```
   This command starts both the Convex backend and the Vite frontend.

3. **Build for Production**:
   ```bash
   pnpm build
   ```

## Folder Structure

- `src/`: Frontend source code
  - `components/`: UI components (Shadcn UI, domain components)
  - `pages/`: Route pages
  - `lib/`: Utilities and constants
- `convex/`: Backend functions and schema
